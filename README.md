# linky - jQuery plugin
**linky** is a simple jQuery plugin to attach links to any element.

## Version
0.2.0

## Usage

	 $(document).ready( function() {
	     $('span').linky( { href: 'http://example.com/', text: 'click me', target: '_blank' });
	 });
	
## Vendors
jQuery - [http://jquery.com](http://jquery.com)
